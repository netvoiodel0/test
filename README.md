# Сайт для тестов

Открой `index.html` в браузере. На сайте есть два раздела:

- `Социология`: 6 тестов по 50 вопросов из `Да.docx`.
- `Политология`: 6 тестов по 50 вопросов из `нет.docx`.

Правильный ответ в документе был первым вариантом каждого вопроса. На сайте вопросы и варианты перемешиваются при загрузке, а правильный ответ показывается сразу после выбора варианта. Если ответить неправильно, вопрос добавляется в конец теста для повторения.

Если нужно будет заменить или добавить вопросы, удобный формат:

```text
Тест 1
1. Текст вопроса?
A) Вариант
B) Вариант
C) Вариант
D) Вариант
Правильный ответ: B
Пояснение: необязательно
```

Можно прислать и Word/текстом/скрином, но обычный текст быстрее всего превратить в готовую базу вопросов.

## Публикация на GitHub Pages

1. Загрузи файлы в корень репозитория: `index.html`, `styles.css`, `app.js`, `questions.js`, `README.md`, `.nojekyll`, `.gitattributes`, `.gitignore`.
2. В GitHub открой `Settings` → `Pages`.
3. В `Build and deployment` выбери `Deploy from a branch`.
4. Выбери ветку `main` и папку `/root`, затем нажми `Save`.
5. Через минуту GitHub покажет ссылку на сайт.

Сайт использует относительные пути, поэтому будет работать и в корне домена, и по адресу репозитория на GitHub Pages.
