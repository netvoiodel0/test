# Сайт для тестов

Открой `index.html` в браузере. Вопросы загружены из `Да.docx`: 3 теста по 100 вопросов.

Правильный ответ в документе был первым вариантом каждого вопроса. На сайте вопросы и варианты перемешиваются при загрузке, а правильный ответ показывается сразу после выбора варианта.

Если нужно будет заменить или добавить вопросы, удобный формат:

```text
Тест 1
1. Текст вопроса?
A) Вариант
B) Вариант
C) Вариант
D) Вариант
Правильный ответ: B
Пояснение: необязательно
```

Можно прислать и Word/текстом/скрином, но обычный текст быстрее всего превратить в готовую базу вопросов.
